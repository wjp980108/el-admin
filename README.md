# 进销存

## 简介

Vue3.0 + Vite + TypeScript + Axios +Element Plus + Pinia + Echarts

## 使用

### 安装依赖

```sh
pnpm install
```

### 运行

```sh
# 开发环境
pnpm run dev
```

### 格式化代码

```sh
# 使用 prettier 规则
pnpm run format
```

### 打包

```sh
# 生产环境
pnpm run build
# 预生产环境
pnpm run build-staging
# 测试环境
pnpm run build-testing
```
